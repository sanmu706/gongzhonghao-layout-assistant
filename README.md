# 公众号排版助手

一个本地优先、可直接部署到 GitHub Pages 的公众号文章排版工具。

## 功能

- 粘贴文章草稿后实时生成公众号排版预览
- 支持视觉风格、标题样式、引用样式、重点文字、列表、分割线等组件化设置
- 支持选中文本后直接套用局部组件样式
- 支持复制 HTML，用于粘贴到微信公众号官方编辑器
- 支持复制纯文本和 Markdown
- 纯前端实现，无需后端服务

## 使用方式

1. 打开页面
2. 在左侧粘贴文章草稿
3. 在中间选择视觉风格和组件样式
4. 选中文章中的文字后，点击对应组件样式即可局部应用
5. 在右侧预览确认效果
6. 点击「复制 HTML」，粘贴到微信公众号官方编辑器

## 在线部署

这个项目是静态页面，最适合部署到 GitHub Pages。

部署步骤见 [DEPLOY.md](./DEPLOY.md)。

## 本地打开

直接用浏览器打开 `index.html` 即可使用。

如果复制 HTML 在本地 `file://` 环境下不稳定，建议部署到 GitHub Pages 后通过 HTTPS 链接使用。

## 隐私说明

文章内容只在浏览器中处理，不会上传到服务器。若部署在 GitHub Pages，页面仍然是纯静态前端应用。
